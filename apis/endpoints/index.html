Environment API Documentation

Returns all environment from a registered account.

graph TD; https://server/api/env/-->GET/; https://server/api/env/-->GET/uuid; https://server/api/env/-->POST/; https://server/api/env/-->DELETE/;

Get

endpoint: /env/

Returns Environments registered on the database.

Parameters

No parameters.

Responses

Code Description
200 Success
400 Bad Request
401 Not Authorized

Example:

GET http://server/api/env/

Response 200 ok
[
 {"uuid":  "string"}
 {"uuid":  "string"}
 {"uuid":  "string"}
]
Consuming the API
  1. Enter in the url http://server/api/env/
  2. Click on GET button at the top right side of the screen
  3. Make sure you are logged in.
